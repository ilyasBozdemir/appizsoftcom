# Ana Sayfa

Bu sayfa, projenizin ana sayfasını temsil eder.

## Kullanım

Bu sayfa, projenizin ana sayfasının içeriğini gösterir.

```jsx
import React from 'react';

function HomePage() {
  return (
    <div>
      <h1>Hoş Geldiniz!</h1>
      <p>Projenizin ana sayfasına hoş geldiniz.</p>
    </div>
  );
}

export default HomePage;
